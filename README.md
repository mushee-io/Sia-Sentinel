# Sia Sentinel

A premium Next.js MVP for a Sia host reliability dashboard, styled for grant submission demos.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Deploy to Vercel

- Push this folder to GitHub
- Import the repo into Vercel
- Deploy

## Notes

- This version uses polished mock data so the UI is ready immediately.
- The landing page includes the supplied Mushee logo as an animated rain background.
- Dashboard, host profiles, incidents, and methodology pages are included.
